# Seekr

**Seekr** — это инструмент для генерации идей и построения дерева идей с помощью LLM (Language Model) на основе Transformers.  
Позволяет создавать иерархические структуры идей, управлять ролями и категориями генерации, а также расширять идеи дочерними узлами.

---

## Основные возможности

- Генерация идей через LLM (поддержка Qwen и Hugging Face моделей).  
- Построение дерева идей с заданной глубиной и шириной.  
- Управление текущей ролью модели (`role`) и категорией генерации (`category`).  
- CLI на базе Typer для удобного взаимодействия.  
- Возможность будущей интеграции TUI и расширений (например, поиск информации по узлам).

## Установка

```bash
# Клонируем репозиторий
git clone https://github.com/username/seekr.git
cd seekr

# Установка зависимостей через pip
pip install -e .
````

**Примечание:** Для работы необходим Python 3.11+ и доступ к локальной модели Hugging Face или интернет для загрузки модели.

## Использование

### CLI команды

```bash
# Просмотр текущей роли модели
seekr role

# Установка новой роли модели
seekr role "ассистент идей"

# Сброс роли к дефолтной
seekr role --reset

# Просмотр текущей категории
seekr category

# Установка новой категории генерации
seekr category "creative"

# Сброс категории к дефолтной
seekr category --reset

# Показать список всех категорий
seekr category --list

# Создание нового дерева идей
seekr idea "Новая идея" --depth 2 --breadth 3

# Добавление дочерних идей к существующему узлу
seekr idea --parent <NODE_ID> --depth 1 --breadth 2
```
